How to export/import a module

Use keyword export in the module before a variable or function:

  •  export const PI = 3.1415926;
  •  export {
      variance,
       mean }
  •  export default stats;

And use keyword import in your main JavaScript file:

  •  import { PI } from './pi.js';
  •  import { mean, variance } from './stats.js';
  •  import stats from './stats.js';
